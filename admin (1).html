إعدادات JSONBin.io

💡 سجل على jsonbin.io وانشئ Bin جديد. خد الـ ID من URL والـ Master Key من صفحة API Keys.

خطوات الإعداد:

  1. سجل على jsonbin.io (مجاني)
  2. اضغط "Create" واختر "Bin"
  3. انسخ JSON الافتراضي من الأسفل والصقه في الـ Bin
  4. خد الـ Bin ID من URL (الجزء بعد /b/)
  5. خد الـ Master Key من صفحة API Keys
  6. اكتبهم فوق واضغط "حفظ"

JSON الافتراضي:

{"products":[]}
